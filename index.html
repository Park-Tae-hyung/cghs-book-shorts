📚 북며드는 쇼츠 제작소

우리 학교 캡컷(CapCut) 프로 계정은 모두 3개! 반별로 정해진 계정을 사용해요.
편집을 시작할 때 「사용 시작」, 끝나면 꼭 「사용 종료」를 눌러 주세요. 🙌
⚠️ 아직 구글 시트가 연결되지 않았어요! (선생님용 안내)
① 새 구글 시트 생성 → [확장 프로그램] > [Apps Script]
② 함께 드린 Code.gs 내용을 붙여넣고 저장
③ [배포] > [새 배포] > 웹 앱 (실행: 나 / 액세스: 모든 사용자)
④ 배포된 웹 앱 URL을 이 HTML 파일 안의 API_URL 에 붙여넣기
현황을 불러오는 중이에요… ⏳
현황은 10초마다 자동으로 새로고침됩니다. (구글 시트 연동)
입력한 반·이름과 사용 기록은 구글 시트에 자동 저장되며 모두에게 공유됩니다.
